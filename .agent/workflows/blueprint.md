# Workflow: Strategic Blueprint (/blueprint)

Este workflow é usado para tomar decisões de alto nível e desenhar a estratégia de novos módulos ou campanhas.

### Passos:

1. **Análise de Dados**:
   - Consulte o `Kommo-MCP` para tendências de vendas e comportamento de leads.
   - Verifique o Dashboard do Looker Studio para métricas de performance (Mirroring).

2. **Desenho de Fluxo**:
   - Esboce o fluxo de agentes necessário para a nova estratégia.
   - Defina se novos campos no CRM ou Zandu são necessários.

3. **Validação**:
   - O Agente de Automação (n8n) deve validar se a infraestrutura aguenta o novo fluxo.

4. **Documentação**:
   - Registre a nova estratégia no `PROJECT_STATUS.md` sob a seção de planejamento.
